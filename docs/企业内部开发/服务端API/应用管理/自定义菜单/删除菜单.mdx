**请求方式：**GET（**HTTPS**）
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/menu/delete?access_token=ACCESS_TOKEN&agentid=AGENTID

**参数说明：**

|  参数 | 必须  | 说明  |
| ------------ | ------------ | ------------ |
| access_token  |  是 | 调用接口凭证  |
| agentid | 是  |  应用id|


**权限说明：**

仅企业可调用；第三方不可调用。

**返回结果：**

```
{
   "errcode":0,
   "errmsg":"ok"
}

```
