**修改待办**
该接口用于修改指定的待办信息，支持修改待办整体状态、待办参与人及其状态。
**请求方式：** POST（**HTTPS**）  
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/todo/update?access_token=ACCESS_TOKEN
**请求包体：**
```json
{
  "todo_id": "17c7d2bd9f20d652840f72f59e796AAA",
  "status": 1,
  "attendees": [
    {
      "userid": "lisi",
      "status": 2
    },
    {
      "userid": "zhangsan",
      "status": 1
    }
  ]
}
```
**仅修改待办整体状态示例：**
```json
{
  "todo_id": "17c7d2bd9f20d652840f72f59e796AAA",
  "status": 0
}
```
**参数说明：**
| 参数 | 是否必须 | 说明 |
| ----------- | -------- | --------------------------------- |
| access_token | 是 | 调用接口凭证。获取方法查看“[获取access_token](https://work.weixin.qq.com/api/doc/90001/90143/90605)” |
| todo_id | 是 | 待办ID |
| status | 否 | 待办整体状态。<br>0 - 完成<br>1 - 进行中 |
| attendees | 否 | 待办参与人列表，最多支持20个参与人。可不传或传空数组；不传或为空时不修改参与人列表 |
| attendees.userid | 是 | 待办参与人ID。当传入attendees时，该字段必填 |
| attendees.status | 是 | 参与人的待办状态。当传入attendees时，该字段必填。0 - 完成<br>1 - 进行中 |
**权限说明：**
- 调用应用的可见范围需要包含根部门。
- 仅允许修改当前应用创建的待办。
- 不允许修改已删除的待办。
**返回结果：**
```json
{
  "errcode": 0,
  "errmsg": "ok"
}
```
