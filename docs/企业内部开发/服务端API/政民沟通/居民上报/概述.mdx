居民上报是企业微信提供的面向微信居民使用的政民沟通工具。微信居民可以在居民群中@机器人、在分享的小程序中、线下扫描二维码多种方式上报问题，上报将交由政府工作人员办理。使用居民上报前，需要配置网格及网格负责人，此外还可配置上报事件类型。
同时，可使用API接口做以下操作：查询网格拉取网格及网格负责人、拉取企业工单概况、拉取个人工单情况、分类统计、拉取工单详情列表、拉取工单详情。
