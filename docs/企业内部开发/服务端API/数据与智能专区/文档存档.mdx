## 概述
企业微信支持应用在数据与智能专区中，检索企业授权的文档。目前该功能仅灰度开放，已灰度企业可前往「企业微信管理后台-安全与管理-管理工具-数据与智能专区-企业文档」使用。
![enter image description here](https://wework.qpic.cn/wwpic3az/215753_lCtU-HIQQOu7AuC_1776825378/0)

## 如何调用
### 1. oauth2授权
在数据专区检索文档，需要用户通过oauth2授权进入应用，若企业管理员已为应用开启了数据与智能专区文档存档权限，则应用调用[获取访问用户身份](#15047)时，企业微信返回user_doc_ticket，可在30天内调用数据专区通用模型时检索该用户的文档。

### 2. 调用通用模型能力
企业管理员先为应用配置通用模型能力的输入协议：
![enter image description here](https://wework.qpic.cn/wwpic3az/136495_vv9hMS9vS4WYb4Q_1776825959/0)
其中，{doc_fragments}变量用于替换检索文档之后的片段数据。

接下来，应用在调用[通用模型能力接口](#56425)时，传入doc_retrieval_params参数（包括关键词），企业微信数据专区使用关键词对用户有权限的文档进行检索，再将检索得到的片段替换上述输入协议的{doc_fragments}变量，然后调用通用大模型。

### 3. 成员选择专区文档
在[通用模型能力接口](#56425)中，应用在doc_retrieval_params参数还可指定docid列表，以指定文档检索范围。应用可通过jsapi[数据与智能专区文档选择](#61186)或小程序接口[数据与智能专区文档选择](#61183)，让用户选择其创建的doc文档，应用即可获得相应文档的docid列表。
