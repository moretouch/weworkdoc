用于删除指定智能文档中的数据表内容块。

::: warning 注意
**仅删除关联**：此操作仅删除智能文档关联的数据表内容块 。
**不可恢复**：删除操作不可撤销，请谨慎操作。
:::


**请求方式**：POST（**HTTPS**）
**请求地址**：https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartdoc/delete_smartsheet?access_token=ACCESS_TOKEN
 
**请求包体**
```json
{
    "docid": "DOC_ID",
    "block_id": "BLOCK_ID"
}
```

**参数说明**
| 参数 | 类型 | 是否必须 | 说明 |
|------|------|----------|------|
| docid | string | 是 | 智能文档的文档 id |
| block_id | string | 是 | 要删除的数据表 Block ID |

**权限说明**
- 自建应用需配置到"[可调用应用](#43883)"列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 第三方应用需具有"文档"权限
- 代开发自建应用需具有"文档"权限

**返回示例**
```json
{
    "errcode": 0,
    "errmsg": "ok"
}
```

**参数说明**
| 参数 | 类型 | 说明 |
|------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
