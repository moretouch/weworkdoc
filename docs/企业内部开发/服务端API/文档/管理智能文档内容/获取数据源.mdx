该接口用于获取智能文档绑定的数据源信息。
支持的数据表接口/回调如下：
| 接口 | 说明 |
| ------|------|
| [添加字段](#53114) | 在数据表里添加字段 |
| [删除字段](#53115) | 在数据表里删除字段 |
| [更新字段](#53116) | 在数据表里更新字段 |
| [查询字段](#53117) | 查询数据表字段列表 |
| [添加记录](#53118) | 在数据表里添加记录 |
| [删除记录](#53119) | 在数据表里删除记录 |
| [更新记录](#53120) | 在数据表里更新记录 |
| [查询记录](#53121) | 查询数据表记录列表 |
| [字段变更事件](#58864) | 数据表的字段变更回调事件 |
| [记录变更事件](#58863) | 数据表的记录变更回调事件 |


**请求方式**：POST（**HTTPS**）
**请求地址**:  https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartdoc/get_smartsheet_info?access_token=ACCESS_TOKEN
 
**请求包体**
```json
{
    "docid": "DOC_ID"
}
```
 
**参数说明**
| 参数     | 类型   | 是否必须 | 说明   |
|--------|--------|------|------|
| docid | string | 是    | 智能文档的文档 id，可通过 [新建文档](#43939) 接口创建 |
 
**权限说明**
- 自建应用需配置到"[可调用应用](#43883)"列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 第三方应用需具有"文档"权限
- 代开发自建应用需具有"文档"权限


**返回示例**
```json
{
    "errcode": 0,
    "errmsg": "ok",
    "ss_docid": "SS_DOC_ID"
}
```

**参数说明**
| 参数       | 类型   | 说明   |
|----------|--------|---------|
| errcode  | int32  | 错误码 |
| errmsg   | string | 错误码说明 |
| ss_docid | string | 绑定的数据表docid。若之前已绑定则返回已有的docid，否则自动创建后返回新的docid |
