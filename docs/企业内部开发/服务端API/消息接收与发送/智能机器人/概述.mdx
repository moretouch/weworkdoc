## API设置
若智能机器人开启API模式，当用户跟智能机器人交互时，企业微信会向智能机器人API设置中的URL的回调地址推送相关消息跟事件。

## 接收回调与被动回复
当用户跟智能机器人交互时，企业微信会向智能机器人的回调URL上推送相关消息或者事件，开发者可根据接收的[消息](#57141)或者[事件](#59058)，[被动回复消息](#59068)。具体流程如下：
