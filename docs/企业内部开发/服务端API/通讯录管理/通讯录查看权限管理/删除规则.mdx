# 删除规则

该接口用于根据规则 ID 批量删除通讯录隐藏规则。

**请求方式：** POST（**HTTPS**）

**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/contactrule/delete?access_token=ACCESS_TOKEN

**请求包体：**
```json
{
  "rule_ids": [10001, 10002]
}
```

**参数说明：**
| 参数 | 是否必须 | 说明 |
| ----------- | -------- | --------------------------------- |
| access_token | 是 | 调用接口凭证。仅通讯录同步应用的 `access_token` 可调用该接口 |
| rule_ids | 是 | 要删除的规则 ID 列表，最多支持 100 个 |

**权限说明：**
- 仅允许通讯录同步应用调用。

**返回结果：**
```json
{
  "errcode": 0,
  "errmsg": "ok"
}
```

**返回参数说明：**
| 参数 | 说明 |
| ----------- | --------------------------------- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
