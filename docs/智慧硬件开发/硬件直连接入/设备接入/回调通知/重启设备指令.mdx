从云端下发重启设备的指令

**指令包体：**

```
{
	"cmd" : "push_restart_device",
	"headers": 
	{
      		"req_id" : "xxxx"
	}
}
```

**参数说明：**

| 参数名       | 类型     | 描述                               |
| --------- | ------ | -------------------------------- |
| req_id    | string | 透传请求的req_id                      |
| cmd       | string | 指令号，此处固定为push_restart_device         |
