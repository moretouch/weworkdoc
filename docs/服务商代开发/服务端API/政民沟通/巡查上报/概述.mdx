巡查上报是企业微信提供的面向成员使用的网格化治理工具。成员可以将日常巡查中发现的问题进行上报，并由网格中心负责人统一分派办理。使用巡查上报前，需要配置网格及网格负责人，此外还可以自定义配置上报事件类型。
同时，可使用API接口做以下操作：查询网格拉取网格及网格负责人、拉取企业工单概况、拉取个人工单情况、分类统计、拉取工单详情列表、拉取工单详情。
