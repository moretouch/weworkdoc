用于删除指定智能文档中的页面。

::: warning 注意
**子页面处理**：删除页面时，该页面下的所有子页面也会被一并删除。
**不可恢复**：删除操作不可撤销，请谨慎操作。
:::


**请求方式**：POST（**HTTPS**）
**请求地址**：https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartdoc/delete_page?access_token=ACCESS_TOKEN
 
**请求包体**

```json
{
    "docid": "DOC_ID",
    "page_id": "PAGE_ID"
}
```
 
**参数说明** 
| 参数 | 类型 | 是否必须 | 说明 |
|------|------|----------|------|
| docid | string | 是 | 智能文档的文档 id |
| page_id | string | 是 | 要删除的页面 ID |
 
**权限说明**
- 自建应用需配置到"可调用应用"列表中的应用secret所获取的accesstoken来调用
- 第三方应用需具有"文档"权限
- 代开发自建应用需具有"文档"权限

**返回示例**
```json
{
    "errcode": 0,
    "errmsg": "ok"
}
```

**参数说明**
| 参数 | 类型 | 说明 |
|------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
