## 旧版jweixin调用
### 示例代码
```
wx.invoke('createExternalPayment', {
	    paymentType:0,  //收款方式
       	totalFee：300，  //收款金额，单位为分
        description:'可乐一罐' //收款说明
     }, 
     function(res){
        if(res.err_msg == "createExternalPayment:ok"){
            paymentId = res.paymentId ; //收款项目id
        }else {
            //错误处理
        }
    });
```
