**接口说明**：调用该接口，可选择一个或多个自己创建的doc文档，选中后返回对应文档的docid与url，应用可在数据与智能专区中使用这些文档调用通用模型能力进行分析（灰度内测中）。

**调用前提：**
> 必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态。
>当前成员必须在应用的可见范围之中，否则调用时会报“no permission”错误。
>应用需具有数据与智能专区文档存档权限，且成员在该应用的文档存档范围内，否则调用时会报“no permission”错误。

### wx.qy.selectDatazoneDoc(Object object)
#### 参数

##### Object object

| 参数名      | 类型       | 必填   | 说明                               | 
| -------- | -------- | ---- | -------------------------------- |  
|success|否|Function|接口调用成功的回调函数|
|fail|否|Function|接口调用失败的回调函数|
|complete|否|Function|接口调用结束的回调函数（调用成功、失败都会执行）|

#### 返回结果

##### Object res
| 参数名      |  类型    | 说明                               | 
| -------- | -------- | -------------------------------- | 
|res|Obj|返回信息|
|res.errMsg|string|执行结果。"wedocSelectDoc:ok" 表示执行成功，否则表示报错信息|
|res.docList|Array<object>|返回的文档列表|
|res.docList[].docid|string|选择的文档docid|
|res.docList[].url|string|选择的文档url|


#### 示例代码

```javascript
wx.qy.selectDatazoneDoc({
	success: function(res) {
		var docList = res.docList;
		for (var i = 0; i < docList.length; i++)
     	{
				var docid = docList[i].docid;
				var url = docList[i].url;
     	}
	},
	fail: function(res) {
		console.log(res)
	},
});
```

|:-----------|:------------|

| qy.selectDatazoneDoc:fail no session or expired |没有调用 wx.qy.login，或者 session_key 已过期 |
