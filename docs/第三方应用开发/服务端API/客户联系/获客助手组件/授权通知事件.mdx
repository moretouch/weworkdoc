企业授权获客助手组件时，会跟安装第三方应用一样产生[授权通知事件](#14951)。
当企业更改授权的获客助手链接时，会回调change_auth事件。此时可调用[获取获客助手授权链接列表接口](#50644)查询当前授权的链接


**请求包体：**
以变更授权通知为例
```
<xml>
    <SuiteId><![CDATA[wxexxx]]></SuiteId>
    <InfoType><![CDATA[change_auth]]></InfoType>
    <TimeStamp>1341673221</TimeStamp>
    <AuthCorpId><![CDATA[CORPID]]></AuthCorpId>
</xml>
```


参数说明：

  | 参数  | 说明 |
  |---- | ----- |
  |SuiteId | 获客助手组件的SuiteId|
  |InfoType | 事件类型：可以为create_auth/change_auth/cancel_auth |
  |TimeStamp | 时间戳 |
  |AuthCorpId | 授权方的corpid，change_auth和cancel_auth事件中返回 |
  |State|构造授权链接指定的state参数，create_auth和change_auth事件中返回|
