# 数据权限授权和到期事件

> 适用于具有「数据权限」的第三方应用 / 代开发应用。数据权限包括：获客助手接口数据、数据专区会话接口数据、数据专区知识集接口数据、数据专区文档接口数据。
>
> 当授权企业的数据权限发生「授权成功」「即将到期」「到期」状态变化时，企业微信服务器会向应用的「指令回调URL」推送对应事件，便于服务商及时刷新应用内状态、感知授权/续期、提前提醒到期。
>
> 一次授权可能涉及多个数据权限项，此时**每个权限项各推送一条事件**。

## 数据权限授权成功事件

具有数据权限的第三方/代开发应用，当授权企业完成某项数据权限的授权（首次授权、到期前续期、到期后重新授权）后，企业微信服务器会向应用的“指令回调URL”推送该事件。

请求示例：
```
<xml>
	<SuiteId><![CDATA[ww4asffe99e54c0f4c]]></SuiteId>
	<AuthCorpId><![CDATA[wxf8b4f85f3a794e77]]></AuthCorpId>
	<InfoType><![CDATA[auth_data_permission]]></InfoType>
	<TimeStamp>1403610513</TimeStamp>
	<AuthType><![CDATA[data_zone_chat]]></AuthType>
</xml>
```

参数说明：

  | 参数 | 说明 |
  |----|---- |
  |SuiteId|第三方应用ID |
  |AuthCorpId|授权企业的CorpID |
  |InfoType|固定为auth_data_permission |
  |TimeStamp|时间戳 |
  |AuthType|数据权限类型，取值见下方“AuthType 取值说明” |

## 数据权限即将到期事件

当授权企业某项数据权限距到期还剩 7 天时，企业微信服务器会向应用的“指令回调URL”推送该事件（到期前仅推送一次）。

请求示例：
```
<xml>
	<SuiteId><![CDATA[ww4asffe99e54c0f4c]]></SuiteId>
	<AuthCorpId><![CDATA[wxf8b4f85f3a794e77]]></AuthCorpId>
	<InfoType><![CDATA[expire_remind_data_permission]]></InfoType>
	<TimeStamp>1403610513</TimeStamp>
	<AuthType><![CDATA[data_zone_chat]]></AuthType>
</xml>
```

参数说明：

  | 参数 | 说明 |
  |----|---- |
  |SuiteId|第三方应用ID |
  |AuthCorpId|授权企业的CorpID |
  |InfoType|固定为expire_remind_data_permission |
  |TimeStamp|时间戳 |
  |AuthType|数据权限类型，取值见下方“AuthType 取值说明” |

## 数据权限到期事件

当授权企业某项数据权限到期（授权被自动取消）时，企业微信服务器会向应用的“指令回调URL”推送该事件。

请求示例：
```
<xml>
	<SuiteId><![CDATA[ww4asffe99e54c0f4c]]></SuiteId>
	<AuthCorpId><![CDATA[wxf8b4f85f3a794e77]]></AuthCorpId>
	<InfoType><![CDATA[expire_data_permission]]></InfoType>
	<TimeStamp>1403610513</TimeStamp>
	<AuthType><![CDATA[data_zone_chat]]></AuthType>
</xml>
```

参数说明：

  | 参数 | 说明 |
  |----|---- |
  |SuiteId|第三方应用ID |
  |AuthCorpId|授权企业的CorpID |
  |InfoType|固定为expire_data_permission |
  |TimeStamp|时间戳 |
  |AuthType|数据权限类型，取值见下方“AuthType 取值说明” |

## AuthType 取值说明

上述三个事件的 `AuthType` 字段取值含义一致，标识发生状态变化的数据权限类型：

  | AuthType | 说明 |
  |----|---- |
  |customer_acquisition_data|获客助手接口数据 |
  |data_zone_chat|数据专区·会话接口数据 |
  |data_zone_knowledge|数据专区·知识集接口数据 |
  |data_zone_document|数据专区·文档接口数据 |
