**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/docdata/get_auth_info?access_token=ACCESS_TOKEN

**参数说明：** 

| 参数      | 是否必须| 类型| 说明   |
| ------- | ---- | --|-- |
|access_token|是|String|调用接口凭证|


**权限说明:**
调用接口的应用需要满足如下的权限：
| 应用类型      | 权限要求 |
| --------  | ---- | 
|自建应用|不支持 |
|代开发应用|不支持| 
|第三方应用|需具备「!!#ff0000 数据与智能专区-分析企业文档数据!!」权限|
> 分析企业文档数据权限需要企业管理员二次授权

**返回结果：**

```
{
  "errcode": 0,
  "errmsg": "ok",
  "status": 1,
  "auth_scope": {
    "userid_list": [
      "xxxx"
    ],
    "department_id_list": [
      1,
      2,
      3
    ],
    "tag_id_list": [
      10,
      11
    ]
  },
  "begin_time": 1726588800,
  "end_time": 1729180799,
  "auth_user_count": 20
}
```
**参数说明：**

|参数|说明|
|:-------|:----------|
|errcode|返回码|
|errmsg|对返回码的文本描述内容|
|status|授权状态，1-试用中；2-试用已结束|
|auth_scope.userid_list|企业授权存档的人员列表|
|auth_scope.department_id_list|企业授权存档的部门列表|
|auth_scope.tag_id_list|企业授权存档的标签列表|
|begin_time|如果授权状态status=1，该字段表示试用开始时间|
|end_time|如果授权状态status=1，该字段表示企业试用到期时间|
|auth_user_count|企业授权存档范围的去重人数|
