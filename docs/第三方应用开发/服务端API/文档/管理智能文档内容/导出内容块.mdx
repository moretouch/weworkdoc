##  提交导出任务
用于提交一个异步任务，以获取指定智能文档的内容。
该接口仅返回任务 ID，实际内容需通过 [查询任务结果](#查询任务结果) 接口轮询获取。
 
**请求方式**：POST（**HTTPS**）  
**请求地址**：https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartdoc/export_task?access_token=ACCESS_TOKEN
 
**请求包体**
```json
{
    "docid": "DOC123456",
    "content_type": 1,
    "url": "https://doc.weixin.qq.com/smartdoc/a1_xxxxxxx",
    "page_id": "abc123"
}
```
 
**参数说明**
 
| 参数 | 类型 | 是否必须 | 说明 |
|------|------|----------|------|
| docid | string | 是 | 智能文档的文档 id |
| content_type | uint32 | 是 | 期望返回的内容格式类型：`1`-Markdown 格式（目前仅支持此格式） |
| page_id | string | 否 | 指定要导出的 Page ID。若提供，则仅导出该 Page 及其所有子 Block 的内容；若不提供，则导出整个文档 |
 
**权限说明**
- 自建应用需配置到"可调用应用"列表中的应用secret所获取的accesstoken来调用
- 第三方应用需具有"文档"权限
- 代开发自建应用需具有"文档"权限

**返回示例**
```json
{
    "errcode": 0,
    "errmsg": "ok",
    "task_id": "task_sp_20260323_abc123"
}
```
 
**参数说明**
| 参数 | 类型 | 说明 |
|------|------|------|
| errcode | int32 | 错误码（0 表示成功） |
| errmsg | string | 错误码说明 |
| task_id | string | 异步任务 ID，用于调用 [查询任务结果](#查询任务结果) 接口轮询任务状态与结果 |

## 查询任务结果
用于通过任务 ID 查询由 [提交导出任务](#提交导出任务) 提交的异步任务状态及结果。当任务完成时，返回智能文档的内容（Markdown格式）。
 
**请求方式**：POST（**HTTPS**）  
**请求地址**：https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartdoc/get_export_result?access_token=ACCESS_TOKEN
 
**请求包体**
```json
{
    "task_id": "task_sp_20260323_abc123"
}
```

**参数说明**
| 参数 | 类型 | 是否必须 | 说明 |
|------|------|----------|------|
| task_id | string | 是 | 异步任务 ID |


**返回示例（任务已完成）**
```json
{
    "errcode": 0,
    "errmsg": "ok",
    "task_done": true,
    "content": "# 产品介绍\n这是导出的 Markdown 内容..."
}
```
 
**返回示例（任务进行中）**
```json
{
    "errcode": 0,
    "errmsg": "ok",
    "task_done": false
}
```

**参数说明**
| 参数 | 类型 | 说明 |
|------|------|------|
| errcode | int32 | 错误码（0 表示成功） |
| errmsg | string | 错误码说明 |
| task_done | bool | 任务是否已完成：`true` - 已完成，`content` 字段有效； `false` - 仍在处理中 |
| content | string | 任务完成时返回的内容 |
