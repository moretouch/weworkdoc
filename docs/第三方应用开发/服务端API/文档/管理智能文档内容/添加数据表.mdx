用于向指定智能文档中添加数据表。
 
**请求方式**：POST（**HTTPS**）

**请求地址**：https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartdoc/add_smartsheet?access_token=ACCESS_TOKEN
 
**请求包体**
```json
{
    "docid": "DOC_ID",
    "info": {
        "title": "数据表标题",
        "after_id": "数据表的block_id"
    }
}
```

**参数说明**
| 参数 | 类型 | 是否必须 | 说明 |
|------|------|----------|------|
| docid | string | 是 | 智能文档的文档 id |
| info | object | 是 | 数据表信息 |
| info.title | string | 否 | 数据表标题 |
| info.after_id | string | 否 | 数据表的block_id。**不支持非数据表类型的block_id** |

**权限说明**
- 自建应用需配置到"[可调用应用](#43883)"列表中的应用secret所获取的accesstoken来调用（[accesstoken如何获取？](#10013/第三步：获取access_token)）
- 第三方应用需具有"文档"权限
- 代开发自建应用需具有"文档"权限

**返回示例**
```json
{
    "errcode": 0,
    "errmsg": "ok",
    "info": {
       "block_id": "BLOCK_ID",
       "title": "数据表标题",
       "sheet_id": "数据表sheet_id",
       "ss_docid": "智能文档docid",
       "after_id": "BLOCK_ID"
    }
}
```
 
**参数说明**
| 参数 | 类型 | 说明 |
|------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| info | object | 添加成功后返回的数据表信息 |
| info.block_id | string | Block ID（系统生成） |
| info.title | string | 数据表标题 |
| info.sheet_id | string | 数据表 sheet_id |
| info.ss_docid | string | 智能文档 docid |
| info.after_id | string | 排序位置（在哪个 Block 之后） |
